# Email Signature

Option 2 brand-forward signature.

Files:
- `signature-option-2.html`: email-ready version using hosted image URLs
- `signature-option-2-preview.html`: local browser preview using local image paths
- `eldar-portrait-bw.png`: black-and-white portrait crop
- `uroboros-logo-green.png`: compact green logo
- `uroboros-logo-chip.png`: compact dark logo chip used in the signature
- `uroboros-logo-black.png`: black logo used next to the name

Notes:
- Email clients do not reliably load custom fonts, so the signature uses `Manrope` first with Arial/Helvetica fallbacks.
- The email-ready file assumes the site is deployed at `https://uroboros.digital`.
- If you want to use a personal LinkedIn profile instead of the company page, replace the LinkedIn URL in the HTML.
